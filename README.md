# Treinamento SOLID

Projeto criado para centralizar as informações do treinamento dos princípios SOLID.

Para esse treinamento, vou utilizar como base o artigo [SOLID: o que é e quais os 5 princípios da Programação Orientada
a Objetos (POO)](https://www.alura.com.br/artigos/solid?srsltid=AfmBOoo1p0B--fdXTLrW_icv7qQnbQaFnJpLoEwfTVIQOokIAQfWOzib)
disponibilizado pela Alura.

A ideia do treinamento é realizar o entendimento dos princípios e criar códigos de exemplo mostrando a problemática 
e a solução.

## FAQ (Perguntas Frequentes)

O direcionamento para cada princípio, vou colocar no arquivo de [FAQ](/docs/faq.md), o qual facilitará encontrar o 
problema e a solução proposta para cada um dos princípios.
