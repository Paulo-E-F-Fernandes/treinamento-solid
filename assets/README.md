# Pasta Assets (Recursos)

É o "baú" de recursos visuais e estáticos do projeto.

A pasta `assets` serve para armazenar todos os arquivos que **apoiam** o texto, mas não são arquivos _Markdown_.

**Organização sugerida:**
* **`images/`**: Prints de tela, fotos e ilustrações usadas nos documentos.
* **`logos/`**: Identidade visual do projeto.
* **`pdf/`**: Documentos para download que não serão editados via _Markdown_.

> **Regra de Ouro:** Nunca use espaços nos nomes de arquivos aqui dentro (ex: use `foto-perfil.jpg` em vez de `foto perfil.jpg`)
para evitar que links quebrem no navegador.
